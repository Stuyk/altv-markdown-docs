# player.rot

**Description**

Set or get the player's rotation.

**Syntax**

```js
public rot: { x: number, y: number, z: number };
```

### Real World Example

```js
const currentRotation = player.rot;
player.rot = { x: 0, y: 0, z: 0 };
```
