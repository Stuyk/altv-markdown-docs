# player.model

**Description**

Set or get the player's model based on string or number.

**Syntax**

```js
readonly model: number | string;
```

### Real World Example

```js
const currentModel = 'mp_f_freemode_01';
player.model = 'mp_f_freemode_01';
player.model = 'mp_m_freemode_01';
```
