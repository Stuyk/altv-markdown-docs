# alt.stopResource

Used to stop a resource by name if it has not already been stopped.

## Usage

```js
alt.stopResource('myresource');
```

## Example Usage

```js
alt.stopResource('myresource');
```

_These examples assume you have imported `alt` from `alt-server`._
