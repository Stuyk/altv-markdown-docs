# alt.removeGxtText

Removes a GXT Text entry.

_There does not seem to be a concise list of all GXT Text Entries_

### Declaration

```typescript
alt.removeGxtText(key: string): void
```

### Usage

```js
alt.removeGxtText('gxtBlipName');
```

### Real World Example

```js
alt.removeGxtText('gxtBlipName');
```
