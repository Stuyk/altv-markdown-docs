# alt.getMsPerGameMinute

Returns milliseconds per game minute.

Returns a `number`;

### Declaration

```typescript
alt.getMsPerGameMinute(): number
```

### Usage

```js
const result = alt.getMsPerGameMinute();
```

### Real World Example

```js
const result = alt.getMsPerGameMinute();
console.log(result);
```

_These examples assume you have imported `alt` from `alt-client`._
